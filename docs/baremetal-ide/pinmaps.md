# Pinmaps

### PMOD Headers

FT header

|            |             |             |
| ---------- | ----------- | ----------- |
| Square Pad | 1 - 1 TDO   | 7 - 5 TDI   |
|            | 2 - 2 nTRST | 8 - 6 TMS   |
|            | 3 - 3 TCK   | 9 - 7 nSRST |
|            | 4 - 4 TXD   | 10 - 8 RXD  |
| GND Row    | 5 - GND     | 11 - GND    |
| VCC Row    | 6 - VCC     | 12 - VCC    |



TileLink header

|            |                 |                   |
| ---------- | --------------- | ----------------- |
| Square Pad | 1 - TL\_CLK     | 7 - CLK\_EXT      |
|            | 2 - TL\_IN\_VAL | 8 - TL\_OUT\_VAL  |
|            | 3 - TL\_IN\_RDY | 9 - TL\_OUT\_RDY  |
|            | 4 - TL\_IN\_DAT | 10 - TL\_OUT\_DAT |
| GND Row    | 5 - GND         | 11 - GND          |
| VCC Row    | 6 - VCC         | 12 - VCC          |
